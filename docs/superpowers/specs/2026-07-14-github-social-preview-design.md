# GitHub Social Preview Design

## Purpose

Create a polished 1280×640 social-preview banner for the `facebook-invites-skill` GitHub repository. The image should explain the workflow at a glance while remaining neutral, professional, and readable at small preview sizes.

## Visual Direction

- Dark-tech aesthetic suited to GitHub.
- Deep navy-to-charcoal gradient background with restrained electric-blue and violet highlights.
- Abstract reactions pop-up represented as a clean interface card, not a screenshot of Facebook.
- A short sequence of generic reaction symbols flowing toward a verified `Invited` state.
- Subtle connection lines or directional motion suggesting the workflow: load, identify, invite, verify.
- Generous negative space and crisp hierarchy.

## Text

Primary title, verbatim:

> Facebook Invites Skill

Secondary line, verbatim:

> Load • Invite • Verify

Typography should be modern, bold, sans-serif, and legible at thumbnail size.

## Composition

- Landscape canvas at 1280×640, matching GitHub's social-preview aspect ratio.
- Title and subtitle on the left half.
- Abstract workflow illustration on the right half.
- Keep important content away from the outer edges to survive preview cropping.

## Constraints

- No Facebook logo or copied Facebook interface.
- No GitHub logo.
- No page names, sports branding, profile photos, usernames, or reaction counts.
- No screenshots, access tokens, API keys, credentials, or personal information.
- No watermark or extra text.
- Avoid excessive glow, visual clutter, tiny interface labels, and stock-photo styling.

## Deliverable

Save the selected PNG as `assets/facebook-invites-social-preview.png`, reference it near the top of `README.md`, commit it to `main`, and push it to the public repository.
